# 🐾 Veterinaria Landing Page

Landing page profesional para clínicas veterinarias, desarrollada con HTML, CSS y JavaScript. Diseño moderno, responsive y optimizado para atraer y convertir visitantes en clientes.

## 📋 Acerca del proyecto

Sitio web estático para servicios veterinarios que incluye:

- **Página principal** con hero section, beneficios, proceso y testimonios
- **Página de servicios** con tarjetas de especialidades y precios
- **Página e nosotros** con valores, historia y diferenciadores
- Diseño **100% responsive** (móvil, tablet, escritorio)
- Paleta de colores profesional (azules corporativos)
- Componentes reutilizables (header y footer cargados dinámicamente)
- Animaciones suaves y microinteracciones

## 🚀 Tecnologías utilizadas

- HTML5 semántico
- CSS3 moderno (variables CSS, Flexbox, Grid, animaciones)
- JavaScript vanilla (para carga dinámica de componentes y menú móvil)
- Diseño mobile-first
